## Config.toml Persistence Issue

**Sorun:** WebUI (`streamlit`) çalışırken `config.toml` sürekli sıfırlanıyor:
- `llm_provider` `"deepseek"` → `"openai"`'a dönüyor
- `deepseek_api_key` siliniyor (boş kalıyor)
- `voice_name` siliniyor

**Kök neden:** `app/config/config.py`'daki `save_config()` fonksiyonu, `_cfg` sözlüğünü diske yazar. Bu sözlük sadece Python'da yüklenen anahtarları içerir. Elle (veya önceki bir patch'le) eklenen `deepseek_api_key`, `deepseek_base_url`, `voice_name` gibi değerler `_cfg`'de yoksa dosyaya yazılmaz ve otomatik silinir.

**Çözüm:** 
- WebUI kapalıyken config'i düzelt
- WebUI başlamadan önce CLI'ı çalıştır
- Tüm Python process'lerini öldür: `taskkill /F /IM python.exe`

**Daha kalıcı çözüm:** `save_config()`'i patch'leyerek tüm mevcut TOML anahtarlarını korumasını sağlamak (şu an yapılmadı).

## Pexels API Yüksek Çözünürlük Seçimi

Varsayılan Pexels sonuçları genelde düşük çözünürlüklü (360x640). 1080p+ için:

```python
def search_best_video(query, min_h=1080):
    url = f"https://api.pexels.com/videos/search?query={query}&per_page=10&orientation=portrait"
    r = requests.get(url, headers={"Authorization": KEY}, timeout=15)
    best, best_h = None, 0
    for v in r.json().get("videos", []):
        for f in v.get("video_files", []):
            h, w = f.get("height", 0), f.get("width", 0)
            if h > w and h >= min_h and f.get("link") and h > best_h:
                best = {"link": f["link"], "h": h, "id": v["id"]}
                best_h = h
    return best
```

## Streamlit 8501 Port Çakışması

WebUI 8501 portunu kullanır. Eğer önce bir test_app.py başlatılmışsa, port çakışması olur. Çözüm:

