# ctrl-shift-ok

**Kısayol:** `Ctrl + Shift + Sol/Sağ Ok`
**Açıklama:** Kelime kelime seç

## Kullanım

Windows'ta bu kısayol ile ilgili işlemi gerçekleştir.

## Notlar

- Sol Ok sola doğru, Sağ Ok sağa doğru kelime kelime seçer.
