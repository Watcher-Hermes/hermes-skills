## Screen Shortcuts
Win + PrtScn = Tam ekranı otomatik PICTURES klasörüne kaydeder.
Alt + PrtScn = Sadece aktif pencere panoya kopyalar.
Win + Shift + S = Sol üstte açılır alan seçimli kesme; önce pano, sonra kaydet.

## Fallback notes
- Alt+PrtScn sonrası pano boşsa PowerShell ile çekim yapılmamış demektir. O zaman aktif pencere boyutunu alıp pyautogui ile o bölgeyi kaydet.
