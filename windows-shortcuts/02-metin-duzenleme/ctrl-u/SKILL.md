# ctrl-u

**Kısayol:** `Ctrl + U`
**Açıklama:** Altı çizili

## Kullanım

Windows'ta bu kısayol ile ilgili işlemi gerçekleştir.

## Notlar

- Bazı Türkçe programlarda bu kısayol `Ctrl + Shift + A` olabilir.
