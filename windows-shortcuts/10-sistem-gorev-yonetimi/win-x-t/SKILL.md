# win-x-t

**Kısayol:** `Win + X + T`
**Açıklama:** Görev Yöneticisi (menü üzerinden)

## Kullanım

Windows'ta bu kısayol ile ilgili işlemi gerçekleştir.

## Notlar

- Bazı dizüstü klavyelerde F tuşları için Fn ile birlikte basılması gerekir.
- Win = Windows logo tuşu.
